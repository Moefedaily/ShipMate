## Development Environment

### Required versions

- **Java (JDK):** 21.0.7
- **Maven:** 3.9.9
- **Node.js:** 22.14.0
- **npm:** 11.4.2
- **Angular CLI:** 20.2.0
- **Docker Desktop:** 28.0.4
- **Docker Compose:** v2.34.0
- **Git:** latest stable

### Verify installation

Run these commands to confirm your setup:

```bash
java -version
mvn -v
node -v
npm -v
ng version
docker -v
docker compose version
git --version
```
